# Уроки по созданию одностраничного сайта


### Landing page

**Лендинг** (от англ. *landing page* — «посадочная страница») — это **одностраничный сайт**, созданный с одной чёткой целью: **превратить посетителя в целевое действие**.

Простыми словами: это специальная страница, на которую пользователь «приземляется» после клика по ссылке из:

- Рекламы (Google Ads, Яндекс.Директ, соцсети)
- Email-рассылки
- Поста в соцсетях
- QR-кода

------

### **Главная цель лендинга:**

Убедить пользователя совершить **одно конкретное действие**:

- Оставить заявку / заказать звонок
- Купить товар (особенно актуально для товаров с горячими акциями)
- Записаться на консультацию или услугу
- Скачать приложение / eBook
- Зарегистрироваться на вебинар
- Подписаться на рассылку

------

### **Ключевые особенности лендинга:**

1. **Одна страница** — вся информация структурирована в пределах одного длинного скролла.
2. **Одна цель** — фокус на единственном призыве к действию (CTA — Call to Action).
3. **Минимум отвлекающих элементов** — нет сложного меню, лишних ссылок, чтобы пользователь не «убежал».
4. **Убедительный контент** — чётко отвечает на вопросы:
   - Что это? (продукт/услуга)
   - Зачем мне это? (выгоды, решение боли)
   - Почему именно это/вы? (преимущества, отличия)
   - Что делать дальше? (явный призыв к действию)
5. **Визуальная поддержка** — качественные фото, видео, инфографика, отзывы, гарантии.

------

### **Когда используют лендинги?**

- **Рекламные кампании** — для повышения конверсии и отслеживания эффективности рекламы.
- **Презентация нового продукта/услуги** — чтобы сфокусировать внимание только на нём.
- **Промоакции, скидки, спецпредложения** — чтобы создать ощущение срочности и уникальности.
- **Сбор лидов (заявок)** — для услуг, где важно сначала получить контакт клиента (стройка, обучение, B2B).
- **Мероприятия** — регистрация на конференцию, вебинар, марафон.

------

### **Структура типичного лендинга:**

1. **Цепляющий заголовок (Headline)** + подзаголовок.
2. **Краткое описание выгоды** (что получит пользователь).
3. **Форма захвата** или кнопка CTA (часто вверху, чтобы было сразу видно).
4. **Блок с преимуществами** (обычно иконки + текст).
5. **Подробное описание продукта/услуги**.
6. **Отзывы, примеры работ, кейсы** (социальное доказательство).
7. **Ответы на частые вопросы (FAQ)**.
8. **Повторный призыв к действию** (ещё одна форма или кнопка).
9. **Футер** (контакты, реквизиты, ссылки на соцсети).

------

### **Лендинг vs многостраничный сайт:**

- **Лендинг** — это **спецназ**: узкая задача, быстрый результат, высокая конверсия.
- **Многостраничный сайт** — это **полноценная армия**: много информации, разделов, о компании, блог и т.д.

------

**Примеры в жизни:**

- Страница, на которую вы попадаете, кликнув на рекламу курса английского в Instagram — это лендинг.
- Страница с акцией «Скидка 50% только сегодня на робот-пылесос» — это лендинг.
- Страница записи на бесплатную диагностику у стоматолога — это лендинг.

------

### **Итог:**

**Лендинг — это продающий одностраничник, который ведёт пользователя по чёткому  пути от интереса к действию, минуя все лишние шаги.** Это мощный инструмент в digital-маркетинге, когда нужно получить быстрые и измеримые результаты.
