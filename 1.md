Чтобы вставить изображение в HTML-страницу, вам нужно использовать тег <img>. Вот пошаговая инструкция:

    Базовая структура HTML-страницы:

html

<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Моя страница</title>
</head>
<body>
    <!-- Здесь будет размещено изображение -->
</body>
</html>

    Вставка изображения:

Добавьте тег <img> в секцию <body>:
html

<img src="/home/user/Загрузки/1.webp" alt="Описание изображения">

    Важные атрибуты:

    src — путь к изображению. В вашем случае это полный путь к файлу.

    alt — альтернативный текст, который отображается, если изображение не загрузилось.

    Дополнительные настройки:
    Можно добавить стили для управления размером и выравниванием:

html

<img 
    src="/home/user/Загрузки/1.webp" 
    alt="Описание изображения"
    width="500" 
    height="300" 
    style="display: block; margin: 0 auto;">

    Важные замечания:

    Убедитесь, что путь к файлу указан верно.

    Браузер должен поддерживать формат .webp. Если нет — конвертируйте изображение в .jpg или .png.

    Для лучшей оптимизации можно использовать относительные пути вместо абсолютных.

Пример полной HTML-страницы:
html

<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Страница с изображением</title>
    <style>
        img {
            max-width: 100%;
            height: auto;
            display: block;
            margin: 20px auto;
        }
    </style>
</head>
<body>
    <h1>Моё изображение</h1>
    <img 
        src="/home/user/Загрузки/1.webp" 
        alt="Главное изображение страницы"
        width="500" 
        height="300">
</body>
</html>

Сохраните этот код в файл с расширением .html и откройте его в браузере. Ваше изображение должно отобразиться на странице.
