

### Туториал по Figma: от установки до первого проекта

#### 1. Установка и вход

- **Веб‑версия:** откройте figma.com в браузере → нажмите «Sign up» (регистрация через email или Google).
- **Десктоп‑приложение:** скачайте с официального сайта figma.com (доступно для Windows и macOS).

**Важно:**

- Для полноценной работы нужна учётная запись.
- Бесплатная версия подходит для личных проектов, платная — для команд.

------

#### 2. Интерфейс: основные элементы

- **Рабочая область** (центр экрана) — место для создания макетов.
- **Панель инструментов** (слева):
  - «Frame» (фрейм) — холст для дизайна;
  - «Rectangle», «Ellipse», «Polygon» — фигуры;
  - «Text» — текст;
  - «Pen» — перо для рисования кривых;
  - «Hand» — перемещение холста;
  - «Zoom» — масштабирование.
- **Панель свойств** (справа): настройки выбранного элемента (размер, цвет, эффекты).
- **Меню** (верх): File, Edit, View, Plugins и др.

------

#### 3. Создание первого макета

**Шаг 1. Создайте фрейм**

1. Выберите инструмент **«Frame»** (F на клавиатуре).
2. Кликните на холсте и растяните прямоугольник — это будет ваш рабочий холст.
3. В панели свойств задайте:
   - размер (например, 1440×1024 px для десктопа);
   - цвет фона (через «Fill»).

**Шаг 2. Добавьте элементы**

- **Фигуры:** выберите «Rectangle» или «Ellipse», нарисуйте на холсте.
- **Текст:** нажмите «T», кликните на холсте, введите текст. В панели свойств настройте шрифт, размер, выравнивание.
- **Изображения:** перетащите картинку из папки на холст или нажмите **Shift+Ctrl+K** → выберите файл.

**Шаг 3. Настройте расположение**

- Выделите элемент → перетащите мышью.
- Используйте **направляющие** (появляются при перемещении) для выравнивания.
- Для точного позиционирования введите координаты X/Y в панели свойств.

------

#### 4. Работа с цветом и эффектами

- **Заливка (Fill):** выберите цвет из палитры или введите HEX‑код (например, `#4CAF50`).
- **Обводка (Stroke):** задайте толщину, цвет и стиль (сплошная, пунктир).
- **Эффекты:** в панели свойств → «Effects»:
  - тень (Drop Shadow);
  - размытие (Layer Blur);
  - подсветка (Inner Shadow).
- **Градиент:** в «Fill» выберите «Linear» или «Radial», настройте цвета и угол.

------

#### 5. Группировка и слои

- **Группировка:** выделите несколько объектов → нажмите **Ctrl+G** (или **Object → Group**).
- **Слои:** панель «Layers» слева показывает иерархию элементов.
  - Перетаскивайте слои для изменения порядка.
  - Скрывайте/показывайте слои через иконку глаза.
- **Маски:**
  1. Создайте фигуру (например, круг).
  2. Поместите поверх неё изображение.
  3. Выделите оба элемента → нажмите **Ctrl+Alt+M** (или «Use as Mask»).

------

#### 6. Сохранение и экспорт

**Сохранить проект:**

- Файл автоматически сохраняется в облаке (Figma — облачный инструмент).
- Для локального сохранения: **File → Save Local Copy** (формат `.fig`).

**Экспорт элементов:**

1. Выделите объект (или группу).
2. В панели свойств найдите раздел **«Export»**.
3. Нажмите **+** и выберите формат:

- **PNG** — для изображений с прозрачностью;
- **JPG** — для фото;
- **SVG** — для векторных иконок;
- **PDF** — для печати или презентаций.

1. Настройте масштаб (1x, 2x и т. п.) и нажмите **«Export»**.

**Совет:** для веб‑проектов используйте **SVG** или **PNG с прозрачностью**.

------

#### 7. Полезные функции

- **Горячие клавиши:**
  - **Ctrl+C / Ctrl+V** — копирование/вставка;
  - **Ctrl+Z** — отмена действия;
  - **Ctrl+G** — группировка;
  - **V** — инструмент выделения;
  - **M** — перемещение холста;
  - **Z** — масштабирование.
- **Авто-лейауты:** упрощают создание адаптивных макетов (панель свойств → «Auto Layout»).
- **Компоненты:** создайте повторяющиеся элементы (кнопки, иконки) → преобразуйте в компонент через **Ctrl+Alt+K**. Изменения в компоненте применятся ко всем его копиям.
- **Стили:** сохраняйте цвета, текст и эффекты как стили для повторного использования.

------

#### 8. Работа с изображениями

- **Масштабирование:** потяните за угол объекта с зажатым **Shift** (чтобы сохранить пропорции).
- **Обрезка:** дважды кликните по изображению → потяните края рамки.
- **Удаление фона:** используйте плагин **«Remove BG»** (требуется API‑ключ).
- **Цветокоррекция:** в панели свойств → «Fill» → настройки экспозиции, контраста, насыщенности.

------

#### 9. Плагины и расширения

- **Как установить:**
  1. Меню **Plugins → Find more plugins**.
  2. Введите название в поиск (например, «Unsplash» для бесплатных фото).
  3. Нажмите **«Install»**.
- **Популярные плагины:**
  - **Unsplash** — загрузка фото;
  - **Iconify** — библиотека иконок;
  - **Content Reel** — генерация текста и изображений;
  - **Export PNG Sequence** — экспорт анимации.

------

#### 10. Советы для новичков

1. **Начинайте с шаблонов:** в меню **File → New** выберите шаблон (например, «Web Design»).
2. **Используйте сетки и направляющие** (View → Layout Grid) для выравнивания.
3. **Создавайте компоненты** для повторяющихся элементов (кнопок, карточек).
4. **Проверяйте экспорт:** откройте файл в браузере или графическом редакторе.
5. **Изучайте горячие клавиши** — это ускорит работу.
6. **Сохраняйте версии:** **File → Version History** позволяет вернуться к предыдущим состояниям проекта.

------

#### 11. Где найти помощь

- **Официальная документация:** help.figma.com.
- **Сообщество:** форум Figma Community, группы в соцсетях.
- **Видеоуроки:** YouTube‑каналы по Figma (поиск по «Figma tutorial»).
- **Шаблоны:** в разделе **Community** на сайте Figma.

------

**Итог:**

1. Зарегистрируйтесь и откройте Figma.
2. Создайте фрейм и добавьте элементы.
3. Настройте цвета, эффекты и слои.
4. Сгруппируйте объекты и создайте компоненты.
5. Экспортируйте проект в нужном формате.
6. Используйте плагины для расширения функционала.
