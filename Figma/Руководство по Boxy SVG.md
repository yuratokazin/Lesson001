
**Boxy SVG** — это векторный графический редактор,  ориентированный на создание и редактирование изображений в формате SVG  (Scalable Vector Graphics). Программа сочетает простоту интерфейса с  функциональностью, необходимой для работы с векторной графикой, и  подходит как для начинающих, так и для профессиональных дизайнеров. en.wikipedia.org +2

## Основные характеристики

**Платформы и версии:**

- веб-версия (работает в большинстве современных браузеров); 

- настольные приложения для Windows (Microsoft Store), macOS, Linux (через менеджеры пакетов или AppImage), Chrome OS (как Chrome App); версия для Raspberry Pi (через Pi-Apps). 

**Лицензия:**

- веб-версия доступна по модели подписки;
- настольные приложения для Windows, macOS и Chrome OS распространяются по бессрочной лицензии;
- версия для Linux бесплатна и обладает полным функционалом. 

**Некоторые возможности:**

- базовые инструменты рисования: перо, кривые Безье, прямоугольники, эллипсы, линии, полигоны; 
- работа с текстом, включая поддержку Google Fonts; 

- манипуляции с объектами: масштабирование, вращение, искажение, группировка, выравнивание; en.wikipedia.org +1

- операции с контурами: объединение, пересечение, вычитание и другие булевы операции; 

- работа с цветом и градиентами: линейные и радиальные градиенты, узоры, прозрачность; поддержка слоёв; редактирование исходного SVG-кода напрямую внутри интерфейса; 

- импорт и экспорт в различных форматах (SVG, SVGZ, PNG, JPG, WebP, PDF, HTML); en.wikipedia.org +1

- интеграция с библиотеками ресурсов: шрифты из Google Fonts, клипы и фотографии из Pixabay, образцы цветов из Color Hunt; 

- трассировка растровых изображений в кривые Безье. 

**Ограничения:**

- отсутствие поддержки анимации; 

ограниченный функционал по сравнению с профессиональными редакторами (например, Adobe Illustrator); нет русского языка в интерфейсе. 

## Особенности

- использует движок Chromium для рендеринга SVG, что обеспечивает совместимость с веб-стандартами; 

поддерживает сохранение цветов, градиентов и узоров в разделе `<defs>` документа SVG для повторного использования; имеет панель свойств для настройки параметров объектов; поддерживает горячие клавиши (например, Ctrl+C/Ctrl+V для копирования/вставки, Ctrl+Z для отмены действия); автосохранение проектов. 

## Сценарии использования

- создание иконок для веб-интерфейсов;
- разработка логотипов;
- создание иллюстраций для веб-сайтов и инфографики;
- быстрое прототипирование;
- создание интерактивной графики с последующей анимацией через CSS или JavaScript. 

## Где начать работу

- веб-версия доступна по адресу boxy-svg.com/app; 

для установки на Windows нужно открыть Microsoft Store, ввести «Boxy SVG» в поиске и нажать «Получить» или «Установить»; 

- на Linux программу можно установить через менеджеры пакетов или Flathub. linuxfordevices.com +1

Для эффективного освоения Boxy SVG полезно изучить горячие клавиши и ознакомиться с концепциями векторной графики, такими как векторные  пути, заливка, обводка и преобразования объектов. Также доступны  онлайн-руководства и обучающие материалы. [sky.pro](https://sky.pro/wiki/digital-art/boxy-svg-obzor-i-vozmozhnosti/) +1



### Туториал по Boxy SVG: от установки до первого проекта

#### 1. Установка

**Варианты для разных ОС:**

- **Windows/macOS:** скачайте установочный файл с официального сайта boxy-svg.com.

- **Linux:** используйте Flatpak (рекомендуется):

  bash

- ```bash
  flatpak install flathub com.boxy_svg.BoxySVG
  ```

**Важно:**

- Для веб‑версии откройте boxy-svg.com/app в браузере.
- Настольные версии имеют больше функций, чем веб‑приложение.

------

Установить Boxy SVG на Debian можно с помощью Flatpak — это  универсальный способ, который подходит для большинства дистрибутивов  Linux, включая Debian. linuxfordevices.com +1

## Установка через Flatpak

**Убедитесь, что Flatpak установлен.** Если он ещё не установлен, выполните команду:

```bash
sudo apt install flatpak
```

**Добавьте репозиторий Flathub**, если он ещё не добавлен:

```
flatpak remote-add --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo
```

**Установите Boxy SVG** с помощью команды:

```bash
flatpak install flathub com.boxy_svg.BoxySVG
```

**Запустите приложение.** После установки вы сможете запустить Boxy SVG из меню приложений или через терминал командой `flatpak run com.boxy_svg.BoxySVG`.

## Важные замечания

- **Архитектура системы.** Boxy SVG доступен для архитектур x86_64 и aarch64 (ARM64). Если ваша  система использует другую архитектуру, установка может быть недоступна. 

- **Обновления.** Flatpak автоматически проверяет и устанавливает обновления для  приложений. Чтобы проверить наличие обновлений, используйте команду `flatpak update`.
- **Права доступа.** Некоторые операции могут требовать прав администратора (sudo).  Убедитесь, что у вас есть необходимые права для выполнения команд.

Если у вас возникнут проблемы с установкой, проверьте, что:

- Flatpak и Flathub обновлены до последних версий.
- Нет конфликтов с другими пакетами или настройками системы.
- Интернет-соединение стабильно, так как для загрузки приложения требуется доступ в сеть.

Для более детальной информации можно обратиться к официальной странице Boxy SVG на [Flathub](https://flathub.org/ru/apps/com.boxy_svg.BoxySVG)

#### 2. Первый запуск и интерфейс

После открытия вы увидите:

- **Панель инструментов** (слева): перо, прямоугольник, эллипс, текст и др.
- **Рабочая область** (центр): холст для рисования
- **Панель свойств** (справа): настройки цвета, обводки, трансформации
- **Меню** (верх): File, Edit, View, Object и др.

------

#### 3. Создание простого изображения (пример: логотип)

**Шаг 1. Настройте холст**

- В меню **View → Canvas Size** задайте размер (например, 200×200 px).
- Выберите цвет фона через **Background Color** в панели свойств.

**Шаг 2. Добавьте базовые фигуры**

1. Выберите инструмент **«Прямоугольник»** (Rectangle).
2. Кликните на холсте и растяните фигуру.
3. В панели свойств:
   - задайте цвет заливки (**Fill**) — например, `#FF5733`;
   - настройте толщину обводки (**Stroke**) — например, 2 px.
4. Добавьте **эллипс** (Ellipse) поверх прямоугольника:
   - выберите инструмент «Эллипс»;
   - нарисуйте круг;
   - выровняйте его по центру через **Object → Align**.

**Шаг 3. Добавьте текст**

1. Выберите инструмент **«Текст»** (Text).
2. Кликните на холсте и введите текст (например, «Logo»).
3. В панели свойств:
   - измените шрифт (Font Family);
   - отрегулируйте размер (Font Size);
   - задайте цвет текста.

**Шаг 4. Сгруппируйте элементы**

1. Выделите все объекты (зажмите Shift и кликайте по ним).
2. Нажмите **Object → Group** (или Ctrl+G).
3. Теперь можно перемещать и масштабировать группу целиком.

------

#### 4. Работа с цветом и градиентами

- **Заливка (Fill):** выберите цвет через палитру или введите HEX‑код (например, `#3366FF`).
- **Градиент:**
  1. Выделите объект.
  2. В панели свойств кликните на иконку градиента рядом с **Fill**.
  3. Выберите тип: линейный или радиальный.
  4. Настройте цвета и направление.
- **Обводка (Stroke):** задайте толщину, цвет и стиль (сплошная, пунктир и т. п.).

------

#### 5. Сохранение и экспорт

**Сохранить проект (SVG):**

- **File → Save** (сохраняет в формате Boxy SVG с метаданными).
- **File → Export → SVG** (чистый SVG‑файл).

**Экспортировать в другие форматы:**

- **PNG:** File → Export → PNG (настройте разрешение).
- **JPEG:** File → Export → JPEG (выберите качество).
- **PDF:** File → Export → PDF (для печати).

**Совет:** для веб‑использования выбирайте **SVG** — он масштабируется без потери качества.

------

#### 6. Полезные функции

- **Горячие клавиши:**
  - Ctrl+C / Ctrl+V — копирование/вставка;
  - Ctrl+Z — отмена действия;
  - Ctrl+G — группировка;
  - Space — перемещение холста (при зажатой клавише).
- **Слои:** панель **Layers** позволяет управлять порядком объектов.
- **Привязка к сетке:** включите **View → Snap to Grid** для точного позиционирования.
- **Превью:** **View → Preview** показывает, как будет выглядеть изображение в браузере.

------

#### 7. Советы для новичков

1. **Начинайте с простых форм** (прямоугольники, круги), затем переходите к кривым Безье.
2. **Используйте слои** для разделения элементов (фон, текст, детали).
3. **Экспериментируйте с градиентами** — они добавляют объём.
4. **Проверяйте экспорт:** откройте SVG‑файл в браузере, чтобы убедиться в корректности.
5. **Изучайте шаблоны:** в меню **File → New from Template** есть готовые примеры.
